# tRFTargets_Precition

## 思路
+ 1.利用靶标数据训练模型  

+ 2.获取全部mRNA序列列表，对于输入的tRF序列进行遍历计算特征向量  

+ 3.将计算好的特征向量代入训练好的模型中进行预测打分，输出得分高的序列
